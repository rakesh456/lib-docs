
|Sr. No.   |  Name | Purpose   | Possible Values  |
|---|---|---|---|
| 1  | allowHierarchy  | Set of hierarchy among the list of values. If hierarchy is allowed set of nested values needs to be passed| Default: False<br/><br/> If set to TRUE set of values as group needs to be passed  |
|  2 | allowNewValue  | Add new element which is not available in list. If no result found an option of “Add” button should appear. | Default: False |
| 3  | canRemoveAll  | Removes all the tags selected|  Default: False<br/><br/> If true user can not remove selected item |
| 4  | maxItemCounter  | Displays value of selected tags for the given item counter  |  Default: 2 |
| 5  |  placeholder | Displays text value that describes expected input  |  Example: "Select country" |
| 5  | searchWithHelper |Each element will have a helper text. Elements can be searched by their name or helper text. | Default: False<br/><br/> If true list help of values needs to passed for each element |
