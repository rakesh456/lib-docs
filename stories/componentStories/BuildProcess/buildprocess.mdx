Steps to generate new build<br/>
1. From command line on the folder where you extracted the zipped source code file, run:<br/>
 &nbsp;&nbsp;  a) **npm install**<br/>
 &nbsp;&nbsp;  b) **npm run build**<br/><br/>
2. This should generate new bundle.css and bundle.js files in
/UI-Lib-release-root-folder/build folder.<br/><br/>
3. Generating build will run test cases first. <br/><br/>
4. To run components locally, you can use command: **npm run start**
