|Sr. No.   |  Name | Purpose   | Possible Values  |
|---|---|---|---|
| 1  | lowerLimit  | Set lower date limit on the date hierarchy | Start date in MM/DD/YYYY format.  |
| 2  | upperLimit  |Set upper date limit on date hierarchy | End date in MM/DD/YYYY format.|
| 3  | disabledList  |List of dates/months/years that are to be shown as enabled on the date hierarchy | Default: Enable all.<br/><br/>For example: ["01/01/2000","02/04/2000","w1/05/2000", "8/01/2000", "q1/2025"]<br/><br/>And disabled only those that are in the list.|
| 4  | height  |Set height of date hierarchy |Default:200px|
| 3  | showQuarter  |Show quarters in the date hierarchy |True and False <br/><br/>Default:true |
| 3  | showWeek  |Show weeks for each month in date hierarchy |True and False <br/><br/>Default:false|
